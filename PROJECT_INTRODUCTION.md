# WebTable 个人网盘系统 - 项目简介

## 项目概述
WebTable 是一个基于 Spring Boot 的全栈个人网盘系统，实现了文件存储、分享、秒传、转存等核心功能。项目采用前后端分离架构，后端使用 Java + Spring Boot，前端使用 Vue.js + Element Plus，数据库使用 MySQL，缓存使用 Redis。

## 技术栈
- **后端框架**: Spring Boot 3.2.5 + Java 17
- **数据持久层**: MyBatis + MySQL
- **缓存**: Redis (Spring Data Redis)
- **安全认证**: JWT + Spring Security Crypto
- **API文档**: SpringDoc OpenAPI 3.0
- **前端技术**: Vue.js 3 + Element Plus + Axios
- **构建工具**: Maven
- **其他工具**: Lombok、Hutool、PageHelper、邮件服务

## 核心功能

### 1. 文件管理功能
- **文件上传**: 支持分片上传、断点续传、秒传功能
- **文件下载**: 支持权限校验和并发下载保护
- **文件管理**: 创建文件夹、重命名、移动、删除、回收站
- **文件预览**: 支持多种格式文件在线预览
- **存储统计**: 实时显示已用存储空间和总空间

### 2. 高级功能
- **秒传机制**: 基于文件MD5哈希值的重复文件检测
- **文件转存**: 用户间文件共享与转存
- **分享系统**: 生成带密码和有效期的分享链接
- **引用计数**: 智能管理文件引用，避免重复存储
- **回收站**: 文件删除恢复和永久删除功能

### 3. 用户系统
- **用户注册/登录**: 邮箱验证码注册、JWT令牌认证
- **权限控制**: 基于角色的访问控制（RBAC）
- **存储配额**: 用户存储空间限制和管理

### 4. 系统特性
- **并发安全**: 引用计数锁机制防止下载时文件被删除
- **权限校验**: 防止越权操作和ID猜测攻击
- **日志记录**: 操作日志记录和审计
- **性能监控**: AOP切面监控接口性能

## 架构设计亮点

### 1. 存储架构优化
- **全局MD5存储**: 所有文件按MD5值统一存储，避免重复
- **逻辑归属分离**: 物理存储与用户逻辑归属解耦
- **引用计数机制**: 主副记录分离，智能管理文件生命周期

### 2. 数据库设计
- **规范化设计**: 用户表、文件表、分享表分离，外键关联
- **性能优化**: 合理索引设计，避免N+1查询问题
- **事务管理**: 关键操作使用事务保证数据一致性

### 3. 安全设计
- **JWT认证**: 无状态认证，支持分布式部署
- **输入验证**: 所有用户输入进行严格验证
- **权限校验**: 接口级别权限控制，防止越权访问
- **防重放攻击**: SQL层防重复操作保护

### 4. 并发处理
- **下载保护**: 引用计数临时锁机制，防止下载中断
- **乐观锁**: 数据库乐观锁处理并发更新
- **异常处理**: 完善的异常处理机制，保证系统稳定性

3. **权限安全**: 多层权限校验，防止越权操作和数据泄露
4. **用户体验**: 秒传、断点续传等功能提升用户体验
