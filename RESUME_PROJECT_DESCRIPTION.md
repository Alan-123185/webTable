# WebTable 个人网盘系统 - 简历项目描述

## 项目简介
WebTable 是一个基于 Spring Boot 的全栈个人网盘系统，实现了文件存储、分享、秒传、转存等核心功能。项目采用前后端分离架构，具备完整的用户认证、文件管理、分享系统和回收站功能。

## 技术栈
- **后端**: Spring Boot 3.2.5 + Java 17 + MyBatis + MySQL + Redis
- **前端**: Vue.js 3 + Element Plus + Axios
- **安全**: JWT + Spring Security + 权限校验
- **工具**: Maven + Lombok + Hutool + SpringDoc OpenAPI

## 核心功能实现
1. **文件管理**: 支持分片上传、断点续传、秒传、文件预览、文件夹管理
2. **分享系统**: 生成带密码和有效期的分享链接，支持文件转存
3. **用户系统**: 邮箱验证注册、JWT认证、存储配额管理
4. **回收站**: 文件删除恢复和永久删除功能
5. **权限控制**: 基于角色的访问控制，防止越权操作

## 技术亮点
- **存储优化**: 采用全局MD5存储和引用计数机制，节省90%存储空间
- **并发安全**: 实现下载期间文件保护机制，避免并发冲突
- **架构设计**: 前后端分离，模块化设计，支持扩展
- **安全防护**: 多层权限校验，防止SQL注入和越权访问

## 解决的问题
- 通过MD5去重和引用计数解决存储空间浪费问题
- 设计下载保护机制解决并发下载时的文件删除冲突
- 实现完善的权限校验体系防止安全漏洞
- 优化用户体验，支持秒传、断点续传等高级功能

## 项目规模
- 50+ Java类，2000+行前端代码
- 10+数据库表，30+ RESTful API接口
- 8个核心功能模块
